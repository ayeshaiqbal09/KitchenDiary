# KitchenDiary

KitchenDiary is a personal recipe management application built using Angular and ASP.NET Core Web API.

## Features

- Add recipes
- View all recipes
- Search recipes
- Edit recipes
- Delete recipes
- Upload recipe photos
- Add cooking tips and notes
- Rate recipes
- Organize recipes using tags

## Tech Stack

### Frontend
- Angular

### Backend
- ASP.NET Core Web API

### Database
- SQLite
- Entity Framework Core
